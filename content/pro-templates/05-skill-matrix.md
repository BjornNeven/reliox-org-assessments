## 2. Proficiency Levels Definition

Use consistent definitions across your organization for fair assessment.

| Level | Name | Description | Practical Definition |
|-------|------|-------------|---------------------|
| **1** | Awareness | Understands the concept; cannot perform | Has received overview training; recognizes when skill is needed; knows who to call |
| **2** | Can Assist | Can help a qualified person | Works alongside skilled technician; handles simple subtasks; learning by doing |
| **3** | Can Perform | Independently competent | Completes work without supervision; meets quality standards; handles routine problems |
| **4** | Can Train | Expert level; develops others | Trains and certifies others; writes procedures; handles complex/unusual situations |

### Level Indicators

| Aspect | Level 1 | Level 2 | Level 3 | Level 4 |
|--------|---------|---------|---------|---------|
| **Supervision** | Cannot perform | Needs direct supervision | Works independently | Supervises others |
| **Problem-Solving** | Recognizes issue | Follows instructions | Solves routine problems | Solves complex/novel problems |
| **Documentation** | Can read | Can follow | Can update | Can create |
| **Quality** | N/A | Requires verification | Self-verified | Verifies others' work |
| **Troubleshooting** | N/A | Assists with data gathering | Diagnoses common issues | Diagnoses complex/unusual issues |

## Appendix A: Quick Reference Card

### Proficiency Levels at a Glance

| Level | Can Do | Cannot Do |
|-------|--------|-----------|
| 1 - Awareness | Explain concept, recognize need | Perform any hands-on work |
| 2 - Can Assist | Help qualified person, handle subtasks | Work unsupervised |
| 3 - Can Perform | Work independently, solve routine issues | Train others, handle complex issues |
| 4 - Can Train | Everything above + train and certify others | N/A |

### Coverage Rules of Thumb

- **Minimum 2 people at Level 3+** for each critical skill
- **Minimum 1 person at Level 4** for each skill category
- **Each shift needs Level 3 coverage** for high-criticality skills
- **No single point of failure** for any critical skill

### Review Schedule

| Review Type | Frequency | Participants |
|-------------|-----------|--------------|
| Individual skill assessment | Quarterly | Supervisor + Employee |
| Team matrix review | Quarterly | Manager + Supervisors |
| Gap analysis update | Quarterly | Manager |
| Succession planning review | Semi-annually | Manager + HR |
| Full skills audit | Annually | All stakeholders |
